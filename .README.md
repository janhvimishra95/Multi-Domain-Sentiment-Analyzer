# 💬 Sentiment Analyzer

A machine learning + deep learning powered sentiment analysis project that classifies text as **positive** or **negative**.

This project includes:

 ✅ Traditional ML model (Logistic Regression + TF-IDF)
 ✅ Deep Learning model (BERT - HuggingFace)
 ✅ FastAPI backend for predictions
 ✅ Streamlit UI for user interaction

---

## 📌 Project Description

The Sentiment Analyzer processes text input and predicts the sentiment behind it.
It is designed to demonstrate end-to-end ML workflow including:

 Data preprocessing
 Model training
 Model evaluation
 API development
 UI integration

---

## ⚙️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/sentiment-analyzer.git
cd sentiment-analyzer
```

---

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

---

### 3. Train the model

#### Option A: Traditional ML

```bash
python scripts/train.py
```

#### Option B: BERT (Deep Learning)

```bash
python scripts/train_bert.py
```

---

### 4. Run the API (FastAPI)

```bash
uvicorn app.main:app --reload
```

Open in browser:

```
http://127.0.0.1:8000/docs
```

---

### 5. Run the UI (Streamlit)

```bash
streamlit run app/ui.py
```

---

## 🚀 API Usage

### Endpoint: `/predict`

**Method:** POST

### Request Example:

```bash
curl -X POST "http://127.0.0.1:8000/predict?text=I love this product"
```

---

### Response Example:

```json
{
  "sentiment": "positive"
}
```
